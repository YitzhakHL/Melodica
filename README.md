# Melodica: Ruidoso y melancólico como el Shoegaze.

Proyecto básico para desarrollo de aplicaciones web con JQuery, bootstrap y PHP.

![Melodica Logo](public/assets/img/melodica-cover.png)

Acerca de **Melodica**:

- Este proyecto intenta hacer lo posible por parecerse a un Framework de PHP, varias de las funciones y definiciones son estan inspiradas o tomadas prestadas de otros Frameworks como Laravel, KumbiaPHP y Symfony.
- No esperen mucho de este proyecto, es un experimento pequeño muy simple y experimental que nació gracias a un proyecto de residencias.
- Este proyecto esta diseñado por alumno universitario, asi que cualquier error, sugerencia o recomendación es bien recibida.
- La visión de este proyecto esta centrado principalmente en el minimalismo y carga rapida de elementos.
